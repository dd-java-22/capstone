@Singleton
class IssueReportServiceImpl @Inject constructor(authRepository: GoogleAuthRepository, webService: SeeSomethingWebService) : IssueReportService(source)

Concrete implementation of IssueReportService that uses Retrofit and Google authentication.

All network calls are executed on a background coroutine scope and exposed as CompletableFutures for convenient use from Java-based ViewModels.
