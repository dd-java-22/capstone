UserViewModel

open class UserViewModel : ViewModel(source)

ViewModel coordinating user sign-in/sign-out and exposing user and error state.

This ViewModel also manages profile updates, including display name, email, and avatar image uploads. All operations delegate to UserService, and results are published to LiveData streams for UI observation.
